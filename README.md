## Where to Drop - Warzone

Simple script that runs on `!ping` or `!p` command that randomly selects a location to drop in Call of Duty - Warzone.

## List of Available Drop Locations

- Array
- Boneyard
- Downtown
- Easter Egg: Just insta drop
- Factory
- Farmland
- Hills
- Hospital
- Lumber
- Military Base
- Park
- Port
- Prison
- Promenade East
- Promenade West
- Salt Mine
- Stadium
- Storage Town
- Summit
- Superstore
- TV Station
- Train Station
